# Система обработки документов EN+

Корпоративная система для автоматической обработки документов с использованием AI (PaddleOCR).

## Функциональность

### ✅ Реализовано

- **Корпоративный строгий UI** - Темный профессиональный дизайн
- **Обработка документов** - Извлечение данных из изображений и PDF
- **Редактирование полей** - Возможность корректировки извлеченных данных
- **Экспорт данных** - JSON, CSV, Excel (xlsx)
- **Триггеры** - Автоматический мониторинг папок для обработки
- **Визуальная подсветка** - Отображение найденных полей на изображении
- **Настройка полей** - Гибкая конфигурация извлекаемых полей

## Установка

### Backend

1. Перейдите в папку backend:
```bash
cd backend
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

**Важно:** При первом запуске PaddleOCR автоматически скачает необходимые модели (около 500MB).

3. Запустите backend:
```bash
python app.py
```

Backend будет доступен на `http://localhost:5000`

### Frontend

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev сервер:
```bash
npm run dev
```

Frontend будет доступен на `http://localhost:5173`

## Использование

### Обработка документа

1. Перейдите на вкладку "Обработка"
2. Загрузите документ (PDF, JPG, PNG)
3. Настройте поля для извлечения
4. Нажмите "Обработать документ"
5. Просмотрите результаты с подсветкой
6. При необходимости отредактируйте значения
7. Экспортируйте результаты

### Настройка триггеров

1. Перейдите на вкладку "Триггеры"
2. Добавьте путь к папке для мониторинга
3. Настройте автоэкспорт
4. Запустите триггер

## Технологии

- **Frontend**: React, TypeScript, MUI, Vite
- **Backend**: FastAPI, PaddleOCR (PaddleX), Python 3.8+
- **OCR Engine**: PaddleOCR Vision Language (GPU/CPU поддержка)
